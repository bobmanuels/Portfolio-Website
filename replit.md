## Project Architecture
### Tech Stack
- **Frontend**: Pure HTML5, CSS3, JavaScript (Vanilla)
- **Fonts**: Google Fonts (Bricolage Grotesque, Montserrat)
- **Server**: Python SimpleHTTPServer on port 5000
- **Deployment**: Static website ready for publishing

### Design System
- **Primary Color**: #321B15 (Dark Brown)
- **Secondary Color**: #ECE5D8 (Cream)
- **Accent Color**: #FF3500 (Orange Red)
- **Typography**: Bricolage Grotesque (headings), Montserrat (body)

### Features
- Responsive navigation with mobile hamburger menu
- Animated hero section with statistics counter
- Portfolio grid with category filtering
- Image slideshow for each project card
- Video ads section with Vimeo embeds
- Testimonials carousel
- WhatsApp floating contact button
- Smooth scroll navigation
- Call-to-action sections
